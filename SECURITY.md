# Security Policy

## Reporting a Vulnerability

If you believe you found a security issue in RDPuzzle, please do not open a public issue.

Email: mrdanielbenzano@gmail.com

Please include:
- RDPuzzle version or commit hash
- Browser and OS
- A description of the issue
- Reproduction steps
- A sample file if it can be safely shared

I will try to acknowledge reports within 7 days.
